---
name: curriculum
trigger: /curriculum
description: Build a multi-session learning path toward any goal
---

When triggered with `/curriculum <goal>`:
1. Ask: "Current level? Time per day/week? Goal — understanding, exam prep, or practical skill?"
2. Research prerequisites and milestones using web_search + Kiwix
3. Generate a week-by-week plan with daily topics, practice days, and milestones
4. Save to /vault/curriculum/<goal>-curriculum.md
5. Offer: "Want to start Day 1 now?"
