# Tools

- **web_search** — general web search via DuckDuckGo
- **http_request** — SearXNG: `GET http://searxng:8080/search?q=<q>&format=json`; Kiwix: `GET http://kiwix/search?pattern=<q>&lang=eng`
- **web_fetch** — fetch any URL for article content
- **file_write / file_read** — save/load files in /vault/
